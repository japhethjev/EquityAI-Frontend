Developer Documentation
API Reference
Everything you need to integrate EquityAI data into your platform. All endpoints return JSON.
Authentication
All API requests (except /api/v1/status) require your API key in the request header.
# Add this header to every request
X-API-Key: eqai_your_key_here

# Base URL
Base: https://equityai-server-production.up.railway.app
GET /api/v1/status API health check · No auth required ▼
Check API availability and coverage statistics. No API key required.
Example Response
{"status":"ok","ngxStocks":146,"usStocks":500,"signalsCached":28,"pricesUpdatedAt":"2026-06-02T15:35:00Z","docs":"Add X-API-Key header to access data endpoints"}
GET /api/v1/stocks All stocks with signals + prices ▼
Returns all stocks with daily AI signals and latest prices.
Query Parameters
marketNGX (default) or US
signalFilter by BUY, HOLD, or WATCH (optional)
Example Request
GET /api/v1/stocks?market=NGX&signal=BUY
X-API-Key: eqai_your_key_here
Example Response
{"status":"ok","count":26,"market":"NGX","stocks":[{"ticker":"ZENITHBANK","signal":"BUY","score":8.2,"price":131.1,"chgPct":0.08,"currency":"NGN"}]}
GET /api/v1/stock/:ticker Full data for a single stock ▼
Returns complete profile for a single stock including signal, live price, fundamentals, and reasoning.
GET /api/v1/stock/ZENITHBANK
X-API-Key: eqai_your_key_here
{"status":"ok","ticker":"ZENITHBANK","signal":"BUY","score":8.2,"reasoning":"Strong ROE 23.5%, P/E 4.3x undervalued, dividend ₦10/share","price":131.1,"currency":"NGN","pe":4.3,"roe":"23.5%","eps":"25.32","div":10,"revenue":"₦4.19T","pat":"₦1.04T"}
GET /api/v1/prices/:ticker Live end-of-day price ▼
Latest end-of-day price for any NGX or US stock. Updated 4:35 PM WAT (NGX) and 9:35 PM WAT (US).
GET /api/v1/prices/MTNN
X-API-Key: eqai_your_key_here
{"status":"ok","ticker":"MTNN","price":820,"chg":5.2,"chgPct":0.64,"currency":"NGN","updatedAt":"2026-06-02T15:35:00Z","source":"EODHD"}
GET /api/v1/report/:ticker Enterprise Full institutional valuation report ▼
Returns the full institutional-grade valuation report including price targets (DCF, DDM, Comparables), bear/base/bull scenarios, risks, catalysts, sentiment, and analyst commentary.
GET /api/v1/report/ZENITHBANK
X-API-Key: eqai_your_key_here
Response Fields
ratingBUY · HOLD · WATCH
priceTarget12m12-month weighted price target
dcfValue · ddmValue · comparablesValueIndividual model outputs
priceTargetBear · priceTargetBullDownside / upside scenario targets
executiveSummaryAI analyst investment thesis
risksArray of key risks with severity ratings
sentimentScore · overallSentimentMarket sentiment analysis (0–100)
POST /api/v1/portfolio Enterprise AI portfolio construction ▼
Constructs an optimised portfolio from a list of tickers using AI signals, fundamentals, and risk profile. Returns weighted allocations, share quantities, and cost breakdown.
Request Body
{
  "tickers": ["ZENITHBANK", "DANGCEM", "GTCO", "SEPLAT"],
  "budget": 5000000,
  "currency": "NGN",
  "riskProfile": "moderate",  // conservative · moderate · aggressive
  "market": "NGX"
}
